# 写作模板库

此目录包含各类网络小说写作模板，帮助您更高效地组织和创作。

## 可用模板

### 📋 [大纲模板](./大纲模板.md)
用于规划故事整体框架、剧情走向和关键事件点。

### 👥 [人物表模板](./人物表模板.md)
用于设定主角、配角的人物信息和关系。

### 📖 [章节模板](./章节模板.md)
用于组织单个章节的写作，包括提纲和正文区域。

## 使用建议

1. **创作前** - 先用大纲模板规划整个故事
2. **人物设定** - 用人物表模板完整描述角色
3. **逐章创作** - 用章节模板编写每一章节
4. **定期检查** - 参考检查报告进行质量把控

---

**提示**: 您可以根据需要修改和扩展这些模板！
