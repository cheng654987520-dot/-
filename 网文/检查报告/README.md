# 检查报告

此目录用于存放故事内容检查、编辑建议和质量评估报告。

## 报告类型

- **内容检查报告** - 检查故事逻辑、设定一致性
- **编辑建议** - 改进建议和修改记录
- **质量评估** - 整体评分和反馈总结
